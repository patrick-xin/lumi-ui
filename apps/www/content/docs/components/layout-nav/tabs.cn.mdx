## 基本用法

```tsx
import { Tabs, TabsListContent, TabsPanel, TabsTab } from "@/registry/ui/tabs";
```

```tsx
<Tabs>
  <TabsListContent>
    <TabsTab>标签 1</TabsTab>
    <TabsTab>标签 2</TabsTab>
  </TabsListContent>
  <TabsPanel>内容 1</TabsPanel>
  <TabsPanel>内容 2</TabsPanel>
</Tabs>
```

## API 参考

### 基础组件

| 组件 | 描述 |
| :--- | :--- |
| [`Tabs`](https://base-ui.com/react/components/tabs#root)  | 将标签按钮组和对应的内容面板组合在一起。渲染一个 `<div>` 元素。
| [`TabsList`](https://base-ui.com/react/components/tabs#list)  | 将各个标签按钮组合在一起。渲染一个 `<div>` 元素。
| [`TabsTab`](https://base-ui.com/react/components/tabs#tab)  | 一个独立的交互式标签按钮，用于切换对应的内容面板。渲染一个 `<button>` 元素。
| [`TabsIndicator`](https://base-ui.com/react/components/tabs#indicator)  | 一个视觉指示器，可以设置样式以匹配当前选中标签的位置。渲染一个 `<span>` 元素。
| [`TabsPanel`](https://base-ui.com/react/components/tabs#panel) | 当对应的标签被选中时显示的面板。渲染一个 `<div>` 元素。

### 复合组件

| 组件 | 描述 |
| :--- | :--- |
| `TabsListContent`  | 一个包装了 `TabsList` 和 `TabsIndicator` 组件的复合组件。提供默认的类似“胶囊”的样式。
