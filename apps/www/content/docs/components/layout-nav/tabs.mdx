## Basic Usage

```tsx
import { Tabs, TabsListContent, TabsPanel, TabsTab } from "@/registry/ui/tabs";
```

```tsx
<Tabs>
  <TabsListContent>
    <TabsTab>Tab 1</TabsTab>
    <TabsTab>Tab 2</TabsTab>
  </TabsListContent>
  <TabsPanel>Content 1</TabsPanel>
  <TabsPanel>Content 2</TabsPanel>
</Tabs>
```

## API Reference

### Primitives

| Component | Description
| :--- | :--- |
| [`Tabs`](https://base-ui.com/react/components/tabs#root)  | Groups the tabs and the corresponding panels. Renders a `<div>` element.
| [`TabsList`](https://base-ui.com/react/components/tabs#list)  | Groups the individual tab buttons. Renders a `<div>` element.
| [`TabsTab`](https://base-ui.com/react/components/tabs#tab)  | An individual interactive tab button that toggles the corresponding panel. Renders a `<button>` element.
| [`TabsIndicator`](https://base-ui.com/react/components/tabs#indicator)  | A visual indicator that can be styled to match the position of the currently active tab. Renders a `<span>` element.
| [`TabsPanel`](https://base-ui.com/react/components/tabs#panel) | A panel displayed when the corresponding tab is active. Renders a `<div>` element.

### Composite Component

| Component | Description
| :--- | :--- |
| `TabsListContent`  | A composite component that wraps the `TabsList` and `TabsIndicator` components. Default `"pill"` like styles.
