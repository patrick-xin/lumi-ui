## Basic Usage

```tsx
import {
  AlertDialog,
  AlertDialogClose,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogContent,
  AlertDialogTitle,
  AlertDialogTrigger,
} from "@/components/ui/alert-dialog";
```
```tsx
export function AlertDialogDemo() {
  return (
    <AlertDialog>
      <AlertDialogTrigger>Show Alert Dialog</AlertDialogTrigger>
      <AlertDialogContent>
        <AlertDialogHeader>
          <AlertDialogTitle>Are you absolutely sure?</AlertDialogTitle>
          <AlertDialogDescription>
            This action cannot be undone. This will permanently delete your
            account and remove your data from our servers.
          </AlertDialogDescription>
        </AlertDialogHeader>
        <AlertDialogFooter>
          <AlertDialogClose>Cancel</AlertDialogClose>
          <AlertDialogClose>Continue</AlertDialogClose>
        </AlertDialogFooter>
      </AlertDialogContent>
    </AlertDialog>
  );
}
```

### Detached triggers
If you need to trigger an alert dialog from a completely different part of the component tree, or if nesting the trigger inside the root is impractical, use `createAlertDialogHandle` to link a `AlertDialogTrigger` to a `AlertDialog` remotely.

```tsx
const deleteAlertHandle = createAlertDialogHandle();
```

<ComponentPreview name="alert-dialog-detached-triggers" />

### Multiple triggers

A single alert dialog can be opened by multiple trigger elements. You can achieve this by using the same handle for several detached triggers, or by placing multiple `<AlertDialogTrigger>` components inside a single `<AlertDialog>`.

```tsx title="Multiple triggers within the Root part"
<AlertDialogRoot>
  <AlertDialogTrigger>Trigger 1</AlertDialogTrigger>
  <AlertDialogTrigger>Trigger 2</AlertDialogTrigger>
  ...
</AlertDialogRoot>
```
```tsx title="Multiple detached triggers"
const demoAlertDialog = createAlertDialogHandle();

<AlertDialogTrigger handle={demoAlertDialog}>Trigger 1</AlertDialogTrigger>
<AlertDialogTrigger handle={demoAlertDialog}>Trigger 2</AlertDialogTrigger>
<AlertDialog handle={demoAlertDialog}>
  ...
</AlertDialog>
```

The alert dialog can also render different content depending on which trigger opened it. This is achieved by passing a payload to the `<AlertDialogTrigger>` and using the function-as-a-child pattern in `<AlertDialog>`.

The payload can be strongly typed by providing a type argument to the `createAlertDialogHandle()` function.

<ComponentPreview name="alert-dialog-render-different-content" />

### Controlled mode with multiple triggers

You can control the alert dialog’s open state externally using the `open` and `onOpenChange` props on `<AlertDialog>`. This allows you to manage the alert dialog’s visibility based on your application’s state. When using multiple triggers, you have to manage which trigger is active with the `triggerId` prop on `<AlertDialog>` and the `id` prop on each `<AlertDialogTrigger>`.

Note that there is no separate onTriggerIdChange prop. Instead, the `onOpenChange` callback receives an additional argument, `eventDetails`, which contains the trigger element that initiated the state change.

## API Reference

### Primitives

| Component | Description |
| :--- | :--- |
| [`AlertDialog`](https://base-ui.com/react/components/alert-dialog#root) | Groups all parts of the alert dialog. Doesn’t render its own HTML element. |
| [`AlertDialogTrigger`](https://base-ui.com/react/components/alert-dialog#trigger) | A button that opens the alert dialog. Renders a `<button>` element. |
| [`AlertDialogPortal`](https://base-ui.com/react/components/alert-dialog#portal) | A portal element that moves the popup to a different part of the DOM.  Renders a `<div>` element.|
| [`AlertDialogViewport`](https://base-ui.com/react/components/alert-dialog#viewport) | A positioning container for the dialog popup that can be made scrollable. Renders a `<div>` element. |
| [`AlertDialogPopup`](https://base-ui.com/react/components/alert-dialog#popup) | A container for the alert dialog contents. Must be used inside `AlertDialogPortal`. Renders a `<div>` element. |
| [`AlertDialogBackdrop`](https://base-ui.com/react/components/alert-dialog#backdrop) | An overlay displayed beneath the popup.  Renders a `<div>` element. |
| [`AlertDialogTitle`](https://base-ui.com/react/components/alert-dialog#title) | A heading that labels the dialog. Renders an `<h2>` element. |
| [`AlertDialogDescription`](https://base-ui.com/react/components/alert-dialog#description) | A paragraph with additional information about the alert dialog. Renders a `<p>` element. |
| [`AlertDialogClose`](https://base-ui.com/react/components/alert-dialog#close) | A button that closes the alert dialog. Renders a `<button>` element. Renders a `<button>` element. |
| `AlertDialogHeader` | Styled container for convenience purpose. Renders a `<div>` element. |
| `AlertDialogFooter` | Styled container for convenience purpose. Renders a `<div>` element. |

### Composite component

| Component | Description |
| :--- | :--- |
| `AlertDialogContent` | Pre assembled component that wraps `AlertDialogPortal`, `AlertDialogBackdrop`, `AlertDialogViewport` and `AlertDialogPopup` with default styles. |


### AlertDialogContent Layout Variants

| Variant | Description |
| :--- | :--- |
| `responsive` | Mobile bottom sheet; Desktop centered modal. |
| `center` | Centered modal with fade-up transition. |
| `top` | Top-aligned modal with fade-down transition. |
| `scrollable` | Fixed header/footer layout for internal scrolling. |
| `stacked` | Scaling and offset support for nested dialogs. |
| `element-outside` | Supports interactive elements outside the popup box. |
