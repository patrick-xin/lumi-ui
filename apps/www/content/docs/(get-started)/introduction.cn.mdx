Lumi UI 为你提供一系列**开箱即用、高度可定制**的 React 组件，并且你能**完整地看到并拥有所有代码**。它的底层由强大可靠的 **[Base UI](https://base-ui.com/)** 驱动，而样式则交给灵活的 **Tailwind CSS** 处理。

## 两种选择，一种自由

每个 Lumi UI 组件都提供**两种形态**，让你在“快速上手”和“深度定制”之间轻松切换：

### 组合组件 (拿来即用)
它们是预先组装好的“完全体”，覆盖了最常见的交互模式。当你需要快速实现一个标准功能时，直接用它就对了。

<ComponentPreview name="combobox-input-inside-popup" />

**适合场景：** 快速搭建原型、开发标准功能，或者需要在整个项目中保持统一的视觉风格。

### 基础组件 (随心搭建)
它们是更原始的“积木块”，将 Base UI 的所有能力都直接暴露给你。当你需要完全掌控组件的结构和交互细节时，它们就是你的最佳搭档。你甚至可以把基础组件和组合组件混搭，创造出独一无二的设计。

<ComponentPreview name="combobox-customization" />

**适合场景：** 实现独特复杂的布局、打造自定义交互，或者现有的组合组件无法满足你的需求。

> **打个比方：** 组合组件就像精装修的房子，拎包入住；基础组件则是毛坯房和全套工具，任你发挥。建议从组合组件开始，当需要更多自由时，再切换到基础组件。

## Lumi是如何设计的

### 无障碍访问？已内置
所有复杂的键盘导航、焦点管理、ARIA 属性和状态逻辑，Base UI 都已为你妥善处理。你无需编写一行相关代码，就能获得出色的无障碍体验。

### 用 Tailwind 轻松改样式
每个组件都支持标准的 `className` 属性。我们沿用了 **shadcn/ui 的 CSS 变量系统**，这意味着你只需要在 `globals.css` 里修改几个变量，就能让所有组件的风格焕然一新。

### 统一的设计语言
- **动画：** 所有组件使用统一的工具类（如 `animate-popup`, `animate-dialog`），保证交互动画协调一致。
- **交互：** 采用宽松的“点击检测”模式，让可点击区域更大、更友好。
- **主题：** 内置多款精心调配的主题，点击 [主题](/docs/themes/canopy) 即可查看。

## 从微调到重构，随你掌控

你可以根据需求，选择任意层级的定制方式：

1.  **快速换肤：** 通过 `className` 添加或覆盖 Tailwind 工具类。
2.  **切换主题：** 修改全局 CSS 变量，一键换色。
3.  **深度定制：** 使用基础组件，从零搭建你想要的任何样式。
4.  **混合创作：** 将基础组件和组合组件像乐高一样拼装起来。

## 一些重要的约定

**命名规律：**
- 根组件：`Combobox`, `Dialog`, `Dropdown`
- 基础组件：`ComboboxInput`, `ComboboxItem`, `DialogPopup`
- 组合组件：`ComboboxInputGroup`, `ComboboxContent`, `DialogContent`

**属性传递：**
所有基础组件都会将其属性（Props）完整地传递给底层的 Base UI 组件，这意味着你可以直接使用 Base UI 的所有功能，同时享受我们为你准备好的样式系统。
