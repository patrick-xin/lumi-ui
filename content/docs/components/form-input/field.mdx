## Use with other form controls

<Callout variant="warning" title="Warning">
  Form controls must have an accessible name in order to be recognized by assistive technologies. `<FieldLabel>` and `<FieldDescription>` automatically assign the accessible name and description to their associated control.
</Callout>

## API Reference

| Component | Description |
| :--- | :--- |
| [`Field`](https://base-ui.com/react/components/field#root) | Groups all parts of the field. Renders a `<div>` element. |
| [`FieldLabel`](https://base-ui.com/react/components/field#label) | An accessible label that is automatically associated with the field control. Renders a `<label>` element. |
| [`FieldControl`](https://base-ui.com/react/components/field#control) | The form control to label and validate. Renders an `<input>` element. You can omit this part and use any Base UI input component instead. For example, `Input`, `Checkbox`, or `Select`, among others, will work with `Field` out of the box. |
| [`FieldDescription`](https://base-ui.com/react/components/field#description) | A paragraph with additional information about the field. Renders a `<p>` element. |
| [`FieldItem`](https://base-ui.com/react/components/field#item) | Groups individual items in a checkbox group or radio group with a label and description. Renders a `<div>` element. |
| [`FieldError`](https://base-ui.com/react/components/field#error) | An error message displayed if the field control fails validation. Renders a `<div>` element. |
| [`FieldValidity`](https://base-ui.com/react/components/field#validity) | Used to display a custom message based on the fieldâ€™s validity. Requires children to be a function that accepts field validity state as an argument. |
